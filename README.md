# InvestIQ 2.0

A production‑ready investment intelligence web app with:

- Real‑time market data
- Scoring engine
- Price charts
- Modular architecture
- Professional UI
- Dark mode

## Setup

1. Add your Finnhub API key in `/services/marketData.js`
2. Open `index.html` in a browser  
3. Or deploy using Vercel / GitHub Pages / Azure Static Web Apps
